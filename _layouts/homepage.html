Kalliope is a modular always-on voice controlled personal assistant designed for home automation.

The concept is pretty simple:

  • choose an order
  • attach actions to this order

Installation

Kalliope can be installed on multiple linux system, and also on small single-board computers like the Raspberry Pi.
